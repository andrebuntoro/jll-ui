# Documentation

---

## Setup

- install npm modules.

```bash
npm install
```

- setup firebase init

  - get your web api key [here](https://console.firebase.google.com/project/jll-service/settings/general) for `firebaseInit.ts`

## Start Server

```bash
npm start
```

---

## Features

#### Admin user

- Grant Admin Role
- Get Cat Breeds
- Get Cats by Breed
- Get Random Cat

#### Non Admin user

- Get Random Cat
